# NVCHAD Config
  - Install Rust Compiler
  - `brew install hashicorp/tap/terraform-ls`
  - Run Mason Install All after rust compiler is installed

## Commands
* `<leader>xx` - TodoTelescope
* `<SHIFT>v` - select whole line
* `y` - copy
* `p` - paste
* `u` - undo
* `<C-r>` - redo
* `:%s/old_text./new_text` - find and replace
* `/` - find
